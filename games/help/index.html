Game Engine Console Commands


..Listed Below..


help
Opens HELP WebPage.

exit
Opens In-Game EXIT Prompt.

exit now
EXITs Program, Without Exit Logging, or Prompt.

cleardata xxxx
Clears All Save Data, requires admin code "xxxx".

clearlog xxxx
Clears All Echo Logs, requires admin code "xxxx".

hide
Closes In-Game Console.
